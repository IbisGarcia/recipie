        <li> Season with sugar, salt and pepper.</li>

    <ul> 
        <li> Tomato </li>
        <li> Peach wedges</li>
        <li> Red onion</li>
        <li> Cider vinegar</li>
        <li> Olive oil</li>
        <li> Sugar </li>
        <li> Salt</li>
        <li> Pepper</li>

    </ul>
